# Postgres.app Setup Instructions

## Step 1: Download & Install

1. Open: https://postgresapp.com/downloads.html
2. Download the latest version
3. Open the .dmg file
4. Drag Postgres.app to Applications folder
5. Open Postgres.app from Applications

## Step 2: Initialize

1. Click the "Initialize" button in Postgres.app
2. You'll see a green indicator - PostgreSQL is now running!

## Step 3: Let me know

Once you see "PostgreSQL 15 is running", come back here and I'll:
- Run the database migrations
- Start your backend
- Get everything working!

## Postgres.app Features

- ✅ Easy start/stop with one click
- ✅ Runs in the background
- ✅ No terminal commands needed
- ✅ Perfect for local development
